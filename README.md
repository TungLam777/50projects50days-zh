## 💻 安装指南

要开始使用本项目，您需要克隆此仓库到本地计算机。请按照以下步骤操作：

1. 打开终端或命令提示符。
2. 输入以下命令：

   ```bash
   git clone https://github.com/TungLam777/50projects50days-zh.git
   ```

3. 进入项目目录：

   ```bash
   cd 50projects50days-zh
   ```

4. 打开您喜欢的代码编辑器，开始探索项目文件。

```
50projects50days-zh/
```
